== Description ==

Webchecker is a tool for monitoring your website status and access times. 
It measures every 5 minutes the access-time and stores it in a database. The user can see these data as graph in his wordpress admin-panel. This plugin uses the external service of <a href="https://webcheck.zanteres.com">https://webcheck.zanteres.com</a> for monitoring the website.

The terms-of-service can be found here: <a href="https://webcheck.zanteres.com/terms-of-service">webcheck.zanteres.com/terms-of-service</a>


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/webchecker` directory,
   or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Webchecker Name screen to configure the plugin
4. If you use it the first time, please <a href="https://webcheck.zanteres.com/wp-login.php?action=register">register for free</a>.
After about 15 minutes you should get the first results.
